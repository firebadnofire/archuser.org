Flashing your install USB

Installation of Arch Linux requires a usb stick of at least 2GB in size or a DVD-R. Most people will prefer a USB drive as it is faster.
I would reccomend a USB 3.0 flash drive if your computer that supports USB 3.0. A computer that does has blue USB ports.

There are two methods I reccomend:
Rufus (Reccomended)
Etcher(Lacks built in SHA256 check, contains analytics, and first-party advertising)

Flash using Rufus:
1. Download Arch Linux from archlinux.org
2. Insert USB into computer
3. Install Rufus from rufus.ie
4. Select USB drive from top dropdown menu
5. Select .iso image by clicking "Select"
6. Click the checkmark betton and compare the SHA256 signature Rufus shows you with the one found here
7. Click "Start" to begin flashing
8. When prompted, remove USB drive, shutdown your computer, reinsert USB drive, and boot into the Arch Linux live USB
Note: Disable secure boot in BIOS and set the drive to the top of the boot order before booting USB drive

Flash using Etcher:
1. Download Arch Linux from archlinux.org
2. Use a a program such as 7zip to check the SHA256 (7zip method found here)
3. Install Etcher from etcher.io or cmd with `winget install balena.etcher`
4. Insert USB drive
5. Select the .iso image by clicking "Flash from file"
6. Select USB drive by clicking "Select target"
7. Click "Flash" to begin flashing drive. 8. When instructed, remove USB drive, shutdown your computer, reinsert USB drive, and boot into the Arch Linux live USB. Note: Disable secure boot in BIOS and set the drive to the top of the boot order before booting USB drive
