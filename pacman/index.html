Using pacman

Pacman is the package manager for Arch Linux and any distro based off it. Syntax is quite simple and is as follows: 

  
   sudo pacman -S package
   sudo pacman -Q package
   sudo pacman -Syu
   sudo pacman -Rns package

   NOTE: flags are case sensitive

Meanings of flags:
-S = sync, will grab packages from repos
-Syu = update repos and sync outdated
-Q = query database to search for package
-Rns = remove package and unused dependencies
-V = print version

Unlike apt and rpm, Arch does not have a package file like .deb and .rpm.
Instead, Arch has the AUR. a collection of user uploaded packages that you, the user, can download and install.

To install from the AUR, you need to copy the git link of the package you want from aur.archlinux.org
Then you can downloaded with git

 
  git clone link
and then you cd into the directory downloaded and then run this to install it:
  
   makepkg -si

However, there are tools to make this simpler. YAY is one of these tools, along with pamac.
To install both, or one of these, run this command:

  
   curl -o install.sh https://archuser.org/pacman/install-yay.sh 
   chmod +x install.sh
   ./install.sh

Syntax for yay is simalar to pacman and is as follows:

  
   yay package
   yay -Syu
   yay -Rns package

First one will look for a package and install it.
Second will update
Third will remove a package
